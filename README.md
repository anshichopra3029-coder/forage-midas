# Midas Core Banking Application

This project implements a Spring Boot application using JPA entities and H2 database support.

## Entities

* Client
* FinancialAdvisor
* Portfolio
* Security

## Technologies

* Spring Boot 3.x
* Spring Data JPA
* H2 Database
* Apache Kafka

## Build

```bash
mvn clean install
```

## Run

```bash
mvn spring-boot:run
```

## Test

```bash
mvn test
```
